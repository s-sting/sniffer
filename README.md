Sniffer для анализа сетевого трафика

Многофункциональный консольный сниффер для перехвата и анализа сетевого трафика. Поддерживает HTTP, HTTPS (с извлечением SNI), DNS и ARP протоколы. Имеет интерактивный режим, режим реального времени (`watch`) и сохраняет результаты в структурированном JSON-формате.

Возможности

- **HTTP сниффинг** — перехват и декодирование HTTP-пакетов (порт 80)
- **HTTPS сниффинг** — извлечение доменов из TLS handshake (SNI)
- **DNS сниффинг** — мониторинг DNS
- **ARP сниффинг** — отслеживание ARP-запросов и ответов
- **Watch-режим** — непрерывный захват с выводом в реальном времени
- **История сессий** — автоматическое сохранение каждого запуска в отдельный JSON-файл с меткой времени
- **Интерактивный CLI** — удобное управление с подсказками и историей команд
